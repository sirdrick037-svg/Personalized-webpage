# Personalized Webpage

## Project Description

This project is a Personalized Webpage created using HTML, CSS, and JavaScript. It allows users to enter their name and age, stores the information using localStorage, and displays personalized messages.

## Features

- Enter name and age
- Save user information using localStorage
- Display a personalized greeting
- Check if the user is an adult using if...else
- Calculate age in months
- Display a motivational quote five times using a for loop
- Responsive and attractive webpage design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- localStorage

## How to Run the Project

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using Live Server or any web browser.
4. Enter your name and age.
5. Click the **Save Information** button.
